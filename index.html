Cmis4owncloud

Here an CMIS adaptor for owncloud build. It will use the Owncloud API, Apache Chemistry and leverage the Android Client for Owncloud as a starting point

Download this project as a .zip file Download this project as a tar.gz file

Welcome to CMIS4Owncloud.

If you want to get started, do the following:

  1. build apache chemistry with maven: http://chemistry.apache.org/opencmis-how-to-build.html
  2. download tomcat server
  3. change the pom.xml to the server path (as shown below)
  4. build the project with maven using mvn clean package
  5. check http://localhost:8080/cmis4owncloud/ for success

<plugin>
<groupId>org.apache.maven.plugins</groupId>
<artifactId>maven-war-plugin</artifactId>
<configuration>
<warName>cmis4owncloud</warName>
<outputDirectory>I:\xampp\tomcat\webapps</outputDirectory>
</configuration>
</plugin>

This will create the template for chemistry and use the file system as repository. This project will then step by setup replace the file repository implementation with calls to the owncloud api. In order to test this, you can use the OpenCMIS Workbench from http://chemistry.apache.org/java/download.html
