Cmis4owncloud

Here an CMIS adaptor for owncloud build. It will use the Owncloud API, Apache Chemistry and leverage the Android Client for Owncloud as a starting point

Welcome to CMIS4Owncloud.

If you want to get started, do the following:

  1. build apache chemistry with maven: http://chemistry.apache.org/opencmis-how-to-build.html
  2. download tomcat server
  3. build the project with maven using mvn clean package
  4. Run \chemistry-opencmis-server\chemistry-opencmis-server-inmemory on tomcat
  5. check http://localhost:8080/chemistry-opencmis-server-inmemory/ for success
  6. change WEB-INF/classes/owncloud.properties to reflect your owncloud instance

This will create the proxy. You can now add any CMIS-Client and navigate owncloud. In order to test this, you can use the OpenCMIS Workbench from http://chemistry.apache.org/java/download.html
